راهنمای راه اندازی VPN برای iOS، Android و Windows

iOS - استفاده از نرم افزار NapsternetV

برای راه اندازی VPN بر روی دستگاه iOS، مراحل زیر را دنبال کنید:

  1. نرم افزار NapsternetV را از App Store دانلود و نصب کنید.
  2. کانفیگ مورد نظر را از کانال تلگرام ربیت پراکسی تهیه کنید.
  3. فایل کانفیگ را در نرم افزار NapsternetV بارگذاری کنید.

    4. برای اینکار وارد تب configs شوید. سپس روی علامت + تاچ کنید. import inpv config file را انتخاب کنید و از حافظه دستگاه کانفیگ خریداری شده را وارد کنید.

  4. پس از بارگذاری کانفیگ، در لیست کانفیگ ها، کانفبگ خریداری شده را انتخاب کنید و سپس در بخش Home دکمه‌ای که وسطش مثلث است را تاچ کنید.

Android - استفاده از HTTP Injector و NapsternetV

برای راه اندازی VPN بر روی دستگاه Android، مراحل زیر را دنبال کنید:

  1. نرم افزار HTTP Injector را از Google Play دانلود و نصب کنید.
  2. کانفیگ مورد نظر را از کانال تلگرام ربیت پراکسی تهیه کنید.
  3. فایل کانفیگ را در نرم افزار HTTP Injector بارگذاری کنید.

    4. برای اینکار روی آیکن پوشه کلیک کنید و import config را تاچ کنید. در این بخش فایل کانفیگی که تهیه کردید و پسوند آن .ehi است را از حافطه انتخاب کنید. نهایتا دکمه start را تاچ کنید.

Windows - استفاده از Bitvise SSH Client

برای راه اندازی VPN بر روی ویندوز، مراحل زیر را دنبال کنید:

  1. نرم افزار Bitvise SSH Client را از سایت رسمی Bitvise دانلود و نصب کنید.
  2. کانفیگ IPv4 مورد نظر را از کانال تلگرام ربیت پراکسی تهیه کنید.
  3. اطلاعات اتصال SSH را از منبع کانفیگ دریافت کنید (آدرس سرور، پورت، نام کاربری و رمز عبور).
  4. نرم افزار Bitvise SSH Client را باز کنید و اطلاعات اتصال SSH را وارد کنید.

    5. برای اطلاعات بیشتر فایل راهنما. را مطالعه کنید

  5. روی دکمه "اتصال" کلیک کنید تا اتصال SSH برقرار شود.

حالا شما می‌توانید از اتصال VPN بر روی دستگاه خود لذت ببرید و امنیت و حریم خصوصی خود را حفظ کنید!
